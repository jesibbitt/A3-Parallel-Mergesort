# Project Information
This Project is a simulation of Parallel Mergesort using MPI in C++. This project utilizes advanced knowledge of parallel algorithm design and can accurately sort any given list in O(log2n) time. It is designed to work with any number of processors between 4 and 24. 

## File Descriptions
The final project is listed under parallelmergesort.cpp. There are other backup files for old implementation ideas that are still listed due to technical problems with local repo location (personal github account used for school server) 

PMergesortOutput.txt contains samples of output using different seeds for the random arrays.
